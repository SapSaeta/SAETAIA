# Revisar borradores de SaetaIA

Úsame cuando quieras revisar y aprobar los borradores de noticias detectados
por el pipeline de ingesta, sin abrir el navegador.

Ejemplos de cuándo activarme:
- "muéstrame los borradores pendientes"
- "¿qué noticias nuevas hay para revisar?"
- "publica el borrador sobre [tema]"
- "rechaza todos los borradores de github releases"
- "aprueba los que sean de investigación"
